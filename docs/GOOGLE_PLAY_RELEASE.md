# Google Play Release Checklist

- [x] Correct Target SDK set.
- [x] Permissions minimized and requested properly (`app.json`).
- [x] Adaptive Icons added.
- [x] `version` and `versionCode` bumped.
- [x] App signed via EAS or local Keystore.
- [x] Proguard / R8 configured (default for Expo).
- [x] No debug credentials or temporary logs remaining.
- [x] No external payment links violating policies.
