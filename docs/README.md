# ResumeSphere AI v1.0

ResumeSphere is a complete AI Career Platform built with React Native and Expo SDK 55.

## Features
- **Resume Builder & AI Optimizer**: Build and optimize resumes with ATS matching.
- **Job Matching Engine**: Evaluate job fit based on current skills.
- **Career Coach**: Receive AI-generated guidance.
- **Interview Prep**: Practice interviews powered by AI.
- **Portfolio Generation**: Export offline HTML portfolios.
- **Cloud Sync**: Offline-first SQLite database that syncs intelligently with the cloud.

Please check the other markdown documents in this folder for detailed overviews.
